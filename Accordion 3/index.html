Why this term

The term accordion is a reference to the musical instrument of the same name, which can expand and contract. The accordion control is a graphical control element comprising a vertically stacked list of items, such as labels or thumbnails. Each item can be "expanded" or "stretched" to reveal the content associated with that item. The accordion control is used in many web applications, including Google Drive, Dropbox, and Microsoft Outlook.
